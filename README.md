# MangoWC ricing template

![photo](https://raw.githubusercontent.com/RIOTLaF/Mango-Template/refs/heads/main/images/desktop.png)

Welcome to my minimalist template for Mango ricing, give a look on what i did:

- Put a wallpaper using **swaybg**
- Organized the project
- Modified visual aspects like borders and blur
- Added Laptop keys

## Other informations:

- Default Browser: [Google Chrome](https://flathub.org/en/apps/com.google.Chrome)
- Default Terminal: [Kitty](https://github.com/kovidgoyal/kitty)
- Default file manager: Nautilus2

> [!NOTE]
> This dotfiles are using [sunsetr](https://github.com/psi4j/sunsetr) to filter blue light
